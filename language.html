Language Detector

This project predicts the language based on word-length. It is somewhat counter-intutive, but is a good showcase of data based categorization.

Purpose

The aim of the project was to explore whether simple statistics can be used to identify language. Instead of using a full dictionary or machine learning library, the program compares the lengths of words found in different texts.

How the Project Works

  1. Basis: calculate word lengths for each and every word from a given text;
  2. Analysis: based on word-length, build distribution of word-lengths;
  3. Normalize Analysis: Convert counts to percentages for a normalized distribution
  4. Comparison: Compare two texts using basic statistical method - Mean Squared Error (MSE)
  5. Prediction: compare test text with sample distribution and make predictions based on smallest difference/distance

Exercise 1 — get_word_lengths

# Takes a string of text and returns a list of how long each word is
# 
def get_word_lengths(text):
    """
    Extract the length of each word in a text.
    
    Args:
        text: String containing the text to analyze
        
    Returns:
        List of integers representing word lengths
    """
    words_list = text.split()
    word_counts = []
    for word in words_list:
        count = int(len(word))
        word_counts.append(count)
    return word_counts

Exercise 6 — mean_squared_error (Advanced Method)

# Compares two texts using Mean Squared Error on their full word-length distributions
# MSE is more reliable than the basic method because it looks at the whole picture,
# not just the mean and median
def mean_squared_error(text1, text2):
    """
    Calculate MSE between percentage distributions of two texts.

    Returns:
        Float representing MSE (smaller = more similar)
    """
    # Get the percentage distribution for each text
    p = percentage_distribution(text1)
    q = percentage_distribution(text2)

    # Both lists need to be the same length to compare them index by index
    # We pad the shorter one with 0.0 (meaning: no words of that length)
    max_len = max(len(p), len(q))
    p = p + [0.0] * (max_len - len(p))
    q = q + [0.0] * (max_len - len(q))

    # Loop through every word length and accumulate the squared differences
    total_square_difference = 0.0
    for i in range(max_len):
        difference = p[i] - q[i]                              # Difference at this word length
        total_square_difference = total_square_difference + difference * difference  # Square it and add

    MSE = total_square_difference / max_len  # Average the total to get the final MSE score
    return MSE

Exercise 7 — Language Predictors

# Advanced predictor: uses MSE to find which reference language is closest
# Whichever language has the smallest MSE distance wins
def predict_language_advanced(sample_text, english_ref, italian_ref, portuguese_ref):
    """
    Predict the language of a sample text using MSE.

    Args:
        sample_text: Text to classify
        english_ref: Reference text in English
        italian_ref: Reference text in Italian
        portuguese_ref: Reference text in Portuguese

    Returns:
        String: "English", "Italian", or "Portuguese"
    """
    english_distance    = mean_squared_error(sample_text, english_ref)
    italian_distance    = mean_squared_error(sample_text, italian_ref)
    portuguese_distance = mean_squared_error(sample_text, portuguese_ref)

    # Find the smallest distance — that's our predicted language
    advanced_distance = min(english_distance, italian_distance, portuguese_distance)

    if advanced_distance == english_distance:
        return "English"
    elif advanced_distance == italian_distance:
        return "Italian"
    elif advanced_distance == portuguese_distance:
        return "Portuguese"
    else:
        pass
    return "Something went wrong"

Reflection

This project showed that languages can have different statistical patterns, even when we only measure word lengths. One interesting finding was that short samples can be misleading, while larger samples give better evidence. The MSE method was more reliable because it compares the full distribution instead of only a small number of summary values.

A limitation is that word length alone cannot capture all the features of a language. In future, I would improve the system by combining word lengths with letter frequencies or n-grams to make the predictor more accurate.
